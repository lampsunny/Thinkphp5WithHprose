# Thinkphp5WithHprose

#### 项目介绍
Thinkphp5 使用 Hprose例子项目

#### 软件架构
Thinkphp5.1
Hprose2.0


#### 安装教程
修改代码中的域名为实际部署域名
example是Hprose例子

#### 使用说明

index跟MyHprose是一组，用于展示传统手动方式调用Hprose服务
app/hprose/HproseServer和TestServer，TestClient是一组，用于展示TP5集成的方式
